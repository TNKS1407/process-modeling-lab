Process Modeling Lab

プロセスデータのモデル化・診断手法を数値計算で体験するインタラクティブ学習ラボです。各シミュレーションは合成データを生成し、NumPy だけで実装されたアルゴリズムを実際に動かして結果を確認できます。

NumPy / pandas 回帰分析 PLS / PCR JIT ソフトセンサー グレーボックス 転移学習
使い方：各カードの「実行」を押すとシミュレーションが走ります（10〜30秒）。結果の画像とともに「結果の見方」が表示されます。アルゴリズムの実装は GitHub のソースコード（process_modeling_core.py）に公開しています。
